## Installation & Deployment 📦
1. Clone/fork this repository  
2. Modify the content of `index.html` (your name, bio, links, etc.)  
3. Add or replace images in `assets/img/`  
4. Update project details in the portfolio section  
5. Push to your GitHub repo named `<your-github-username>.github.io` (or any name)  
6. Enable GitHub Pages in repo Settings → Pages → Source: main branch → / (root) → Save  

Your site will be live at:  
https://**your-username**.github.io (if repo name = your-username.github.io)  
or https://**your-username**.github.io/**repo-name**
